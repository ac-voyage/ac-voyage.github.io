# Graph

# 图的基本概念

设图 G=⟨V,E⟩G = \langle V, E \rangleG=⟨V,E⟩.

# 子图(subgraph)

G′=⟨V′,E′⟩,V′⊆V,E′⊆EG' = \langle V', E' \rangle, V' \subseteq V, E' \subseteq EG′=⟨V′,E′⟩,V′⊆V,E′⊆E 为图 GGG 的子图.

# 诱导子图(induced subgraph)

G′=⟨V′,E′⟩,V′⊆V,E′={(u,v)∣u,v∈V′,(u,v)∈E}G' = \langle V', E' \rangle, V' \subseteq V, E'=\{(u,v)|u,v\in V',(u,v)\in{E}\}G′=⟨V′,E′⟩,V′⊆V,E′={(u,v)∣u,v∈V′,(u,v)∈E} 是图 GGG 的诱导子图.

# 团(clique)

图 GGG 的一个子图 G′=⟨V′,E′⟩G' = \langle V', E' \rangleG′=⟨V′,E′⟩， 且 G′G'G′ 为关于 V′V'V′ 的完全图.

团数记为 ω(G)\omega(G)ω(G).

# 极大团(maximal clique)

一个团是极大团当且仅当他不是其他团的子集.

# 最大团(maximum clique)

点数最多的团

# 最小染色(minimum coloring)

用最少的颜色给点染色使相邻点颜色不同. 色数记为 χ(G)\chi(G)χ(G)

# 最大独立集(maximum independent set)

最大的一个点的子集使任意两个点不相邻，记为 α(G)\alpha(G)α(G).

# 最小团覆盖(minimum clique cover)

用最少个数的团覆盖所有的点，κ(G)\kappa(G)κ(G).

# 几个性质

  • 团数小于等于色数 ω(G)≤χ(G)\omega(G) \le \chi(G)ω(G)≤χ(G)
  • 最大独立集数小于等于最小团覆盖数 α(G)≤κ(G)\alpha(G) \le \kappa(G)α(G)≤κ(G)
