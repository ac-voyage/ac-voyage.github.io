# 性质一

莫比乌斯函数是积性函数.

μ(a)μ(b)=μ(a⋅b),a⊥b\mu(a)\mu(b) = \mu(a\cdot{b}), \quad a \bot b μ(a)μ(b)=μ(a⋅b),a⊥b
