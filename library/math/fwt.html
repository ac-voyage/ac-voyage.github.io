# 快速沃尔什变换

快速沃尔什变换核心解决的问题是， 对于两个长度为 nnn、从 000 开始编号的数组 AAA 和 BBB， 在 O(nlog⁡n)\mathcal{O}(n\log{n})O(nlogn) 的时间内求出下述定义的数组 CCC

C[i]=∑x⊕y=iA[x]⋅B[y]C[i] = \sum_{x \oplus y \, = \, i}{A[x] \cdot B[y]} C[i]=x⊕y=i∑​A[x]⋅B[y]

其中 ⊕\oplus⊕ 为位运算 and⁡,or⁡,xor⁡\operatorname{and}, \operatorname{or}, \operatorname{xor}and,or,xor 中的一个 ，记为

C=A∗BC = A * B C=A∗B

接下来 +++ 表示算术加法，⋅\cdot⋅ 表示算术乘法. 如果运算符两边是数组，则表示下标对应的两项间做运算.

和 FFT 类似，为了解决这个问题，构造一个变换 F\mathcal{F}F 出来使得

C=A∗B⟺F{C}=F{A}⋅F{B}C = A * B \Longleftrightarrow \mathcal{F}\{C\} = \mathcal{F}\{A\} \cdot \mathcal{F}\{B\} C=A∗B⟺F{C}=F{A}⋅F{B}

同时要存在一个逆变换 F−1\mathcal{F}^{-1}F−1 使得

A=F−1{F{A}}A = \mathcal{F}^{-1}\{\mathcal{F}\{A\}\} A=F−1{F{A}}

前人已经推出了 F\mathcal{F}F 是怎样一个变换. 设 A,BA, BA,B 的长度为 2k2^k2k，不足 2k2^k2k 的部分用 000 补全. 用 A0A_0A0​ 表示 AAA 的前 2k−12^{k-1}2k−1 项（即下标最高位为 000 的项)， A1A_1A1​ 表示 AAA 的后 2k−12^{k-1}2k−1 项（即下标最高位为 111 的项）. [A,B][A, B][A,B] 表示 A,BA, BA,B 拼接而成的新数组.

  • 异或

F{A}=[F{A0}+F{A1},F{A0}−F{A1}]\mathcal{F}\{A\} = \left[\mathcal{F}\{A_0\}+\mathcal{F}\{A_1\}, \mathcal{F}\{A_0\}-\mathcal{F}\{A_1\}\right] F{A}=[F{A0​}+F{A1​},F{A0​}−F{A1​}]

F−1{A}=[F−1{A0+A12},F−1{A0−A12}]\mathcal{F}^{-1}\{A\} = \left[ \mathcal{F}^{-1}\{\frac{A_0+A_1}{2}\}, \mathcal{F}^{-1}\{\frac{A_0-A_1}{2}\} \right] F−1{A}=[F−1{2A0​+A1​​},F−1{2A0​−A1​​}]

  • 按位与

F{A}=[F{A0}+F{A1},F{A1}]\mathcal{F}\{A\} = \left[\mathcal{F}\{A_0\}+\mathcal{F}\{A_1\}, \mathcal{F}\{A_1\}\right] F{A}=[F{A0​}+F{A1​},F{A1​}]

F−1{A}=[F−1{A0}−F−1{A1},F−1{A1}]\mathcal{F}^{-1}\{A\} = \left[ \mathcal{F}^{-1}\{A_0\}-\mathcal{F}^{-1}\{A_1\}, \mathcal{F}^{-1}\{A_1\} \right] F−1{A}=[F−1{A0​}−F−1{A1​},F−1{A1​}]

  • 按位或

F{A}=[F{A0},F{A1}+F{A0}]\mathcal{F}\{A\} = \left[\mathcal{F}\{A_0\}, \mathcal{F}\{A_1\}+\mathcal{F}\{A_0\}\right] F{A}=[F{A0​},F{A1​}+F{A0​}]

F−1{A}=[F−1{A0},F−1{A1}−F−1{A0}]\mathcal{F}^{-1}\{A\} = \left[ \mathcal{F}^{-1}\{A_0\}, \mathcal{F}^{-1}\{A_1\}-\mathcal{F}^{-1}\{A_0\} \right] F−1{A}=[F−1{A0​},F−1{A1​}−F−1{A0​}]

void FWT(int a[],int n)  
{  
    for(int d=1;d<n;d<<=1)  
        for(int m=d<<1,i=0;i<n;i+=m)  
            for(int j=0;j<d;j++)  
            {  
                int x=a[i+j],y=a[i+j+d];  
                a[i+j]=(x+y)%mod,a[i+j+d]=(x-y+mod)%mod;  
                //xor:a[i+j]=x+y,a[i+j+d]=(x-y+mod)%mod;  
                //and:a[i+j]=x+y;  
                //or:a[i+j+d]=x+y;  
            }  
}

void UFWT(int a[],int n)  
{  
    for(int d=1;d<n;d<<=1)  
        for(int m=d<<1,i=0;i<n;i+=m)  
            for(int j=0;j<d;j++)  
            {  
                int x=a[i+j],y=a[i+j+d];  
                a[i+j]=1LL*(x+y)*rev%mod,a[i+j+d]=(1LL*(x-y)*rev%mod+mod)%mod;  
                //xor:a[i+j]=(x+y)/2,a[i+j+d]=(x-y)/2;  
                //and:a[i+j]=x-y;  
                //or:a[i+j+d]=y-x;  
            }  
}
void solve(int a[],int b[],int n)  
{  
    FWT(a,n);  
    FWT(b,n);  
    for(int i=0;i<n;i++) a[i]=1LL*a[i]*b[i]%mod;  
    UFWT(a,n);  
}
